# Photography Portfolio Project

A responsive photography portfolio website layout optimized using CSS visual rules, typography settings, and box-model properties. This project focuses on refining raw HTML with clean, maintainable presentation rules.

## Core Concepts Practiced
* **CSS Visual Rules:** Formatting text alignment, font weights, line heights, and background images.
* **The Box Model:** Managing element spacing using padding, borders, and margins.
* **Layout Design:** Structuring a clean, scannable gallery layout for desktop viewing.

## How to View
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
